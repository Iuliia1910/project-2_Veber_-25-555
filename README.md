# project-2_Veber-M25-555

Это второй учебный проект: простая реляционная база данных на Python с хранением данных в JSON-файлах. 
Поддерживает создание таблиц, добавление, выборку, обновление и удаление данных.  

---

## Установка

### Сначала установите зависимости через Poetry:

poetry install
Или с использованием Make:
make install

### Запуск
Запуск интерактивного интерфейса базы данных:
poetry run project

### После запуска вы получите приглашение к вводу команд:

>>>Введите команду:
Основные команды
Работа с таблицами
Команда	Описание
create_table <имя_таблицы> <столбец1:тип> <столбец2:тип> ...	Создать таблицу
list_tables	Показать список всех таблиц
drop_table <имя_таблицы>	Удалить таблицу (требуется подтверждение)
insert into <имя_таблицы> values (<значения>)	Добавить запись
select from <имя_таблицы> [where <столбец>=<значение>]	Выбрать записи с условием или все записи
update <имя_таблицы> set <столбец>=<значение> where <столбец>=<значение>	Обновить запись
delete from <имя_таблицы> where <столбец>=<значение>	Удалить запись (требуется подтверждение)
info <имя_таблицы>	Показать информацию о таблице: столбцы и количество записей
help	Показать справку
exit	Выйти из программы

## Пример работы
### Запуск базы и создание таблицы:

>>> create_table users name:str age:int is_active:bool
Таблица "users" успешно создана со столбцами: ID:int, name:str, age:int, is_active:bool
Добавление записей:

>>> insert into users values ("Simone", 20, true)
Запись с ID=1 успешно добавлена в таблицу "users".
>>> insert into users values ("Alex", 25, false)
Запись с ID=2 успешно добавлена в таблицу "users".

### Выборка записей:

>>> select from users
+----+--------+-----+-----------+
| ID |  name  | age | is_active |
+----+--------+-----+-----------+
| 1  | Simone | 20  |   True    |
| 2  | Alex   | 25  |   False   |
+----+--------+-----+-----------+

### Обновление записи:

>>> update users set age=21 where name="Simone"
Запись с ID=1 успешно обновлена.

### Удаление записи:

>>> delete from users where ID=2
Вы уверены, что хотите выполнить "удаление записи"? [y/n]: y
Запись успешно удалена.

### Информация о таблице:

>>> info users
Таблица: users
Столбцы: ID:int, name:str, age:int, is_active:bool
Количество записей: 1

### Удаление таблицы:

>>> drop_table users
Вы уверены, что хотите выполнить "удаление таблицы"? [y/n]: y
Таблица "users" успешно удалена.
